# Survivor Tasks (ZomboidTodo)

A Project Zomboid mod prototype that adds an in-game task notebook.

Open a notebook or journal, jot down what you were doing, then tick it off once you survive long enough to finish it.

## Features

- Open the task board from a `Notebook`, `Journal`, or `Diary` inventory context menu
- Add tasks via text entry
- Toggle complete status
- Delete tasks
- Saves per character via `ModData`
- Requires a pen/pencil to edit tasks (viewing is allowed without one)

## Install

Copy the `ZomboidTodo` folder into your Project Zomboid `mods` directory, or use the repo folder as your mod root.

## Usage

1. Start or load a character.
2. Open your inventory.
3. Right-click a `Notebook`, `Journal`, or `Diary`.
4. Select `Open Survivor Tasks`.

## Notes

- This is a prototype/MVP. Expect iteration and breaking changes.
- If you change Mod IDs or save keys during development, existing task lists may not migrate cleanly.
