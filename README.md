# Survivor Tasks

A Project Zomboid mod prototype for an immersive in-game task notebook.

## MVP features
- Open the task board from a notebook/journal inventory context menu
- Add tasks with text entry
- Toggle complete status
- Delete tasks
- Save tasks per character using ModData
- Requires a pen/pencil to modify tasks, but the list can still be viewed without one

## Install
Copy the `ZomboidTodo` folder into your Project Zomboid `mods` directory or use the workspace folder as your mod root.

## Usage
1. Start a character.
2. Open your inventory.
3. Right-click a `Notebook`, `Journal`, or `Diary` item.
4. Select `Open Survivor Tasks`.

You can only add, toggle, or delete tasks when the character has a pen or pencil in inventory.
